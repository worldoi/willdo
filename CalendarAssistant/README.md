# Will do / CalendarAssistant

Will do 是一款面向日常学习与生活场景的智能日历助手。核心目标是把“识别信息 -> 生成日程 -> 同步提醒 -> 实时展示”做成一条顺滑链路，让日程录入和管理尽量自动化。

## 软件支持的功能

### 1) 智能识别与导入

- 文本识别：输入自然语言，自动提取时间、地点、标题等结构化信息
- 截图识别：通过无障碍截图后进行 OCR + AI 解析，快速落地为事件
- 图片识别：导入图片内容识别日程信息
- 多模态识别：结合文字与图像上下文，提高复杂场景识别成功率

### 2) 日程管理能力

- 单次日程：支持新增、编辑、完成、归档
- 重复日程：支持父事件/实例语义、单次修改、排除某次
- 归档体系：过期事件自动归档，支持恢复和清理

### 3) 系统日历同步

- 与系统日历双向同步
- 支持单次与重复事件同步
- 包含映射维护、去重与回环保护，减少重复写入和脏数据

### 4) 通知与胶囊展示

- 日程胶囊：显示即将开始或进行中的关键日程
- 取件聚合胶囊：多个取件/取餐事件聚合展示
- OCR 进度与结果胶囊：识别过程中可视化反馈

> 原生实况通知（Live Activity）能力说明以 Android 16（API 36）为基线；低版本设备按普通通知或厂商适配路径展示。

### 5) 悬浮交互与快捷入口

- 悬浮窗：可在任意界面快速呼出输入与识别能力
- EdgeBar 侧边栏：手势唤起，和悬浮窗协同使用
- 下拉磁贴：快速触发识别日程/悬浮窗功能

### 6) 规则引擎与事件呈现

- 内置规则：日程、列车、打车、航班、取件、取餐、取票、寄件等
- 规则驱动展示：按事件类型套用模板、图标、动作按钮
- 统一渲染入口：减少硬编码分支，便于后续扩展规则

### 7) 平台与系统适配

- Shizuku/Root 辅助能力（用于部分系统权限恢复场景）

### 8) 稳定性与工程能力

- 异常日志捕获与轻量 ANR 监测
- Room 主存 + 旧数据迁移兼容
- 架构边界守卫（`checkArchitectureGuardrails`）

## 基本信息
