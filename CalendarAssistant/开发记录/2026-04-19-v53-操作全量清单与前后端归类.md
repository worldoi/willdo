# 2026-04-19 v53 操作全量清单与前后端归类

## 1. 说明

- 本文是当前仓库的“操作盘点”与“前后端归类”落地稿。
- 归类规则采用已确认版本：
  - 前端操作：仅改变 UI 展示/交互状态，不改业务事实。
  - 后端操作：改变业务事实、持久化、或触发外部副作用，必须 API 化（center-port-node）。
  - 混合操作：UI 触发 + 后端 API 提交。

## 2. 前端操作（允许页面内实现）

### 2.1 Home/页面交互

- Tab 切换（Today/Note/All）、侧边栏开关与页面跳转。
- 搜索模式开关、搜索词输入、搜索栏显隐与焦点切换。
- 日程项滑动 reveal/收起、局部按钮展开。
- 课表展开/收起、拖拽中间态、回弹动画。
- 弹窗显隐：新增/编辑、删除确认、权限提示、遮罩层。

### 2.2 Settings/设置交互

- 各设置页的下拉展开、Sheet 弹出、确认框显隐。
- AI 设置页输入焦点与空白点击清焦点。
- 备份页导入模式（追加/覆盖）临时选择状态。

### 2.3 Floating/悬浮窗交互

- 输入框草稿、日期时间选择器临时状态。
- 编辑态切换、删除确认弹层。
- 滑动手势偏移、动画状态。

### 2.4 全局 UI 临时状态

- Snackbar/Toast/Loading 显隐。
- 列表滚动位置、局部展开态。

## 4. 混合操作（统一模式）

- 模式：UI 触发 -> 后端 API -> 返回结果 -> UI 展示。
- 典型例子：保存按钮、通知动作按钮、导入确认按钮、同步开关按钮。
- 禁止：页面直接写 DAO/Repository 或直接触发外部副作用。

## 5. All 页面展示规则（已落地）

代码位置：`app/src/main/java/com/antgskds/calendarassistant/ui/page_display/AllEventsPage.kt`

- 不展示便签（NOTE）。
- 不展示重复父事件（`isRecurringParent == true`）。
- 展示重复子实例与普通事件。
- 统一限制展示窗口：`startDate <= today + 7 天`。

## 6. API 化实施优先级

### P0

- 日程域 API 收口（增删改、完成/检票/撤销、归档/恢复）。
- 提醒/同步/胶囊副作用出口统一。

### P1

- 课程域与重复实例 this/future/all 操作 API 化。
- 导入导出与识别入库 API 化。

### P2

- 设置域签名标准化（context/source/traceId）。
- Receiver/Tile/Floating 入口统一改为 API 调用。
